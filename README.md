# Portfolio Project

A React + Vite project built with TypeScript.

## Tech Stack

- **React + Vite**
- **TypeScript**
- **Tailwind CSS**
- **CSS**

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd portfolio-project

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```
